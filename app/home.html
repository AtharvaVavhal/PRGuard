Capabilities
Everything your team needs
to ship clean code.
🤖
AI Code Review
Scores every PR 0–10 across 8 categories including naming, complexity, security, and dead code.
💬
Inline Comments
Issues are posted directly on the affected line in the diff — no digging through reports.
🔥
Auto-Fix Branch
When a PR fails, PRGuard creates a fix branch with AI-applied corrections automatically.
📊
Team Dashboard
Live dashboard tracking score trends, pass rates, and issue categories across all repos.
⚙️
Custom Rules
Add a prguard.yml to your repo to define team-specific coding standards.
💬
PR Chat
Comment /prguard <question> to ask the bot anything about your review.
Workflow
From PR to review
in seconds.
01
PR Opened
Developer opens or pushes to a Pull Request on GitHub.
02
Diff Fetched
PRGuard fetches the diff and your team's custom rules from prguard.yml.
03
AI Reviews
LLaMA 3.3 reviews the code and generates a score with detailed issues.
04
Comments Posted
Inline comments and a summary are posted directly on the PR.
05
Auto-Fixed
If score is below threshold, a fix branch is created automatically.
06
Gate Set
Commit status is set to ✅ or ❌ blocking or allowing the merge.
